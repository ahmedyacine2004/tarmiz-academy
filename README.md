# React Learning Repo

This repository is created **purely for learning React**.  
Its only purpose is to practice React concepts, experiment with components, and explore the ecosystem in a simple environment.

## What This Repo Is For

- Understanding how React works
- Practicing components, props, and state
- Learning hooks like useState, useEffect, useContext
- Trying small projects and experiments
- Getting familiar with Vite, TypeScript, and modern tooling

## What This Repo Is NOT For

- Not a production application
- Not optimized for deployment or performance
- Not intended for real-world architecture or business logic

## Tooling

This project uses:

- **React** for UI
- **TypeScript** for type safety
- **Vite** for fast dev environment

No advanced configs, no heavy setups. Just a clean space to learn.

## Goal

The main goal is to help understand the fundamentals of React in a simple and controlled environment, without distractions or extra complexity.
